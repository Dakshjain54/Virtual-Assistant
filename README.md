# 🤖 Virtual Assistant

This project is a **Python-based Virtual Assistant** that can perform various tasks using voice or text commands.

The assistant is designed to automate everyday activities such as searching the web, opening applications, telling the time, and performing other useful tasks.

---

## 🚀 Features

- 🎤 Voice command recognition
- 🌐 Search information on the internet
- 📖 Wikipedia search
- 🎵 Play music
- ⏰ Tell current time and date
- 📂 Open applications and websites
- 💬 Basic conversation responses

---

## 🛠 Tech Stack

- Python
- SpeechRecognition
- pyttsx3 (Text-to-Speech)
- Wikipedia API
- Webbrowser module
